# Huaxia Basketball Manager (华夏篮球联赛模拟器) 🏀

一款以华夏篮球联赛为背景，基于LLM模拟的真实篮球经理游戏。

## 游戏简介

在这款游戏中，你将成为一支球队的总经理，通过球员训练、交易和外援市场来完善球队阵容，带领球队征战整个赛季，目标是夺取总冠军！

### 游戏特色

- 🏆 20支球队可供选择
- 🤖 真实的AI模拟（需配置API）
- 📅 完整的42场常规赛 + 季后赛体验
- 🏋️ 球员训练系统，提升球员能力
- 🔄 球员交易系统
- 🌍 外援市场系统
- 🤕 真实的伤病系统
- 💾 存档/读档功能

## 安装指南

### 第一步：安装 Python

Python 是运行本游戏所需的软件环境。

1. 打开浏览器，访问 Python 官网下载页面：
   ```
   https://www.python.org/downloads/
   ```

2. 点击黄色的 "Download Python 3.x.x" 按钮下载安装程序（推荐python 3.12）

3. 运行下载的安装程序，**注意以下关键设置**：
   - ✅ 勾选底部的 "Add Python to PATH"（非常重要！）
   - 点击 "Install Now" 开始安装

4. 等待安装完成，点击 "Close" 关闭安装程序

5. 验证安装是否成功：
   - 按键盘 `Win + R` 打开运行窗口
   - 输入 `cmd` 并按回车，打开命令提示符
   - 输入以下命令并按回车：
     ```
     python --version
     ```
   - 如果显示类似 `Python 3.12.x` 的版本号，说明安装成功

### 第二步：下载游戏

1. 从 GitHub 下载游戏文件（点击绿色的 "Code" 按钮，选择 "Download ZIP"）
2. 解压下载的压缩包到你想要的位置，例如：`D:\华夏篮球联赛模拟器`

### 第三步：安装游戏依赖

1. 按 `Win + R`，输入 `cmd`，按回车打开命令提示符

2. 进入游戏目录（将下面的路径替换为你的实际路径）：
   ```
   > D:
   > cd D:\华夏篮球联赛模拟器
   ```

3. 安装依赖包：
   ```
   pip install -r requirements.txt
   ```

4. 等待安装完成（可能需要几分钟）


### 第四步：配置API密钥

1. 推荐使用 DeepSeek API（价格便宜，效果好）：

- 访问 DeepSeek 开放平台：https://platform.deepseek.com/
- 注册账号
- 充值（几块钱就够玩很久）
- 在 "API Keys" 页面创建一个新的 API 密钥
- 复制生成的密钥（以 `sk-` 开头的字符串）

2. 用记事本或其他文本编辑器打开游戏目录下的 `config.py` 文件，找到以下内容：

```python
class LLMConfig:
    # API密钥 - 优先从环境变量读取
    API_KEY: str = ""
```

3. 将你的 API 密钥填入引号中：

```python
    API_KEY: str = "sk-你的密钥内容"
```

保存文件后即配置完成。

### 第五步：启动游戏

在游戏目录下，运行以下命令启动 Web 服务器：

```
python -m src.web.app
```

看到类似以下提示说明启动成功：
```
* Running on http://0.0.0.0:5000
```

打开浏览器，访问：
```
http://localhost:5000
```

你将看到游戏主界面，可以开始新游戏或加载存档！

### 第六步：球队和阵容配置（可选）

经过以上五步，游戏已经可以正常游玩。当前游戏默认是20支虚拟球队，每支球队拥有13~14位虚拟球员。用户可以通过tools文件夹下的工具，进行自定义球队配置和自定义球员编辑。工具使用说明可见./tools/README.md

## 游戏玩法简介

### 赛季流程

1. **常规赛**：每支球队打42场比赛
2. **季后赛**：排名前12的球队进入季后赛
   - 四分之一决赛（7战4胜）
   - 半决赛（7战4胜）
   - 总决赛（7战4胜）

### 日常操作

- **比赛**：模拟比赛
- **训练**：训练球员提升能力
- **交易**：与其他球队交易球员或签约自由球员
- **外援**：通过球探搜索外援，提高球队综合实力

## AI模拟功能的说明

> 游戏默认使用本地算法，**不配置AI也能正常游玩**， 但体验感会大幅下滑。
> 配置AI后可获得更真实的模拟数据和更智能的交易评估&外援搜索。

### 功能对比

| 功能 | 使用AI | 不使用AI（本地算法） |
|------|--------|---------------------|
| 比赛模拟 | ✅ 智能叙述、真实数据 | ⚠️ 基础随机算法 |
| 交易评估 | ✅ 考虑球队需求、战术 | ⚠️ 简单价值对比 |
| 随机性 | ✅ 自然随机性 | ⚠️ 固定概率 |

### 模拟时长说明

游戏采取并发api请求模式，因此玩家球队每场and其它球队每轮比赛模拟时长大概在40s~1min，请耐心等待。


### 支持的API服务

| 服务商 | API地址 | 模型名称 |
|--------|---------|----------|
| DeepSeek（推荐） | https://api.deepseek.com/v1 | deepseek-chat |
| 通义千问 | https://dashscope.aliyuncs.com/compatible-mode/v1 | qwen-turbo |
| 其他兼容OpenAI格式的API | 自行配置 | 自行配置 |

如需更换服务商，修改 `config.py` 中的 `BASE_URL` 和 `MODEL` 即可。


## 局域网联机访问

如果你想在其他设备（如手机、平板）上访问游戏：

1. 确保其他设备与运行游戏的电脑在同一个网络下

2. 在运行游戏的电脑上查看本机IP地址：
   - 按 `Win + R`，输入 `cmd`，按回车
   - 输入 `ipconfig` 并按回车
   - 找到 "IPv4 地址"，通常是 `192.168.x.x` 格式

3. 在其他设备的浏览器中访问：
   ```
   http://192.168.x.x:5000
   ```
   （将 `192.168.x.x` 替换为你的实际IP地址）

## 常见问题

### Q: 提示"python不是内部或外部命令"怎么办？

说明 Python 没有正确添加到环境变量。请重新安装 Python，确保勾选 "Add Python to PATH" 选项。

### Q: 提示"No module named xxx"怎么办？

说明依赖包没有安装完整。请在游戏目录下重新运行：
```
pip install -r requirements.txt
```

### Q: Web界面无法访问怎么办？

请检查以下几点：
1. 确保 Web 服务器已启动（运行 `python -m src.web.app`）
2. 检查命令行是否显示 "Running on http://0.0.0.0:5000"
3. 尝试访问 http://127.0.0.1:5000

### Q: AI模拟不工作怎么办？

1. 检查 `config.py` 中的 API_KEY 是否正确填写
2. 确认 API 账户余额是否充足
3. 检查网络连接是否正常

即使 AI 不可用，游戏也会自动使用本地算法，不影响正常游玩，但游戏体验会下降。
