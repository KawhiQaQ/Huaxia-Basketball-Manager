# 华夏篮球联赛 球员自定义配置工具

独立于游戏主体的小工具，用于创建自定义的 `players.json` 和 `teams.json` 文件。

## 功能特点

- 可视化界面，支持自定义20支球队
- 每支球队可添加任意数量球员（建议12人以上）
- 可自动计算球员总评能力值（该功能默认不启用，不启用情况下能获得更自由的球员设置）
- 支持导入现有数据进行编辑
- 一键导出，直接用于游戏

## 使用方法

### 启动工具

```bash
# 在项目根目录运行
python tools/roster_creator/roster_creator.py
```

或者双击 `run_roster_creator.bat` (Windows)

### 操作流程

1. **创建球队**: 点击"+ 添加球队"，填写球队名称、城市、状态
2. **编辑球员**: 选择球队后点击"+ 添加球员"，填写球员信息
3. **导出数据**: 文件 → 导出，选择目录

### 导入现有数据

1. 点击导入按钮
2. 选择players.json或teams.json其中之一，工具会自动搜索文件夹内另一个同时载入（因此请确保当前文件夹内两个文件都存在）
3. 配置球员能力前请先点击球员框右上角的解锁，否则默认锁定无法修改。

### 使用导出的数据

将导出的 `players.json` 和 `teams.json` 复制到游戏的 `player_data` 目录，即可开始使用自定义名单。

## 球员属性说明

| 属性 | 说明 | 范围 |
|------|------|------|
| 进攻 (offense) | 进攻能力 | 1-99 |
| 防守 (defense) | 防守能力 | 1-99 |
| 三分 (three_point) | 三分投射 | 1-99 |
| 传球 (passing) | 传球能力 | 1-99 |
| 篮板 (rebounding) | 篮板能力 | 1-99 |
| 体力 (stamina) | 体力值 | 1-99 |
| 总评 (overall) | 综合能力 | 自动计算 |
| 交易指数 | AI交易意愿 | 1-100 |

## 总评计算公式

```
总评 = 进攻×0.25 + 防守×0.2 + 三分×0.15 + 传球×0.15 + 篮板×0.15 + 体力×0.1
```

勾选"自动计算"时系统自动更新，取消勾选可手动设置，推荐手动自由设置。

## 注意事项

- 球队数量固定为20支（华夏篮球联赛标准）
- 建议每队配置12名以上球员
- 外援球员需勾选"外援"选项，每支球队最多4名外援
- 技能标签最多选择3个
